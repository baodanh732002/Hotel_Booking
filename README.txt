## Các bước chạy Project

# Khởi động MongoDB-Compass:

    - Tải về MongoDB-Compass
    - Mở MongoDB Compass và điền vào mục URL với đường dẫn: mongodb+srv://52000197:khachsan@cluster0.5tcowba.mongodb.net/test
    - Chọn Connect để kết nối với databases

# Thiết lập project

    - Tải project về từ đường dẫn: https://gitlab.duthu.net/S52000511/cuoiky.git
    - Mở project bằng vscode
    - Nhập npm i vào terminal để tải lại node-module
    - Tải xong. Nhập lệnh npm start để chạy project
    - Project sẽ được chạy với cổng 3016
    - Mở đường dẫn http://localhost:3016 để vào trang web
